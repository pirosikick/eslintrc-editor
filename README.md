# eslintrc-editor

https://pirosikick.github.io/eslintrc-editor/

## Development

```
# clone this repository
$ git clone https://github.com/pirosikick/eslintrc-editor.git
$ cd eslintrc-editor

# start on http//localhost:3000
$ npm install && bower install
$ npm start
```
